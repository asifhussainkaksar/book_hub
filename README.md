# book_hub
User Authentication: Implemented a login feature to manage multiple user accounts.
Google Books API: Integrated with the Google Books API to fetch and display book details (cover, title, author, etc.).
Interactive User Interface: Designed with HTML, CSS, and JavaScript to create a user-friendly and responsive interface.
Backend Development: Utilized Node.js and Express.js to build the server-side logic.
Database Management: Used PostgreSQL for storing user data, book information, reviews, and ratings.
